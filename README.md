 SAEP Estoque Facil
 
   ## Descricao
   
    Sistema simples desenvolvido no Hackathon SAEP para cadastrar produtos em um banco de dados utilizando Python, Tkinter e MySQL.
    
  ## Situacao-problema
   Uma pequena loja precisa controlar seus produtos, pois atualmente os registros sao feitos em papel. Isso causa erros, perda de informacoes e dificuldade para consultar o estoque.
    
  ## Tecnologias utilizadas
    - Python
    - Tkinter
    - MySQL
    - VS Code

  ## Funcionalidade da Aula 01
    - Criar banco de dados
    - Criar tabela de produtos
    - Conectar Python ao MySQL
    - Criar tela inicial 
    - Cadastrar produto no banco de dados
